Create a Candle (class)

An inventory for my home-made candles.

Sept 27, 2015

Start by creating a class for Candle. (With a capital C for Candle.) All of my candles have a fragrance, a color, a size and a shape. Also, since there isn’t much use to an inventory quantities, I want to be able to keep track of how many I have, and if that changes.

When you create a class, you start with an object method that is constructor. In Ruby, this is “initialize.” When you initialize your class, you create parameters that you can set equal to instance variables. Each of these instance variables defines a single attribute that every object in the class will have.

Now we have a class called Candle. Every time we make a new batch of candles, we know what attributes they are going to have.

To get a pleasant, easy-to-read output of your information, define the "inspect" method to suit your purposes. This example below overrides the default “inspect".

There are a lot of methods we can put into this class, but this has gone on for too long already, so lets wrap up with what happens if we make more candles, or decide to burn one (or more) of the candles we already have.

In order to make this work you need to set up reading and writing methods.

Add the attr_accessor line inside the Class, above the initialize statement. You can set them up for any instance variable. If you only want to allow permission to read your variable, just use attr_reader, likewise, if you only want to allow it to be written use attr_writer. These methods are also called getter and setter methods respectively.

What we have done so far is pretty easy, but just think of all the ways this info can be used and transformed!
