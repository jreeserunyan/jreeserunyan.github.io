Hello version control

What, how and why!

Aug 24, 2015

Version control is beneficial because it keeps things tidy. Especailly with open source projects, where there are endless versions of things, developers would be lost without a good version control system.

Git allows you to create commits, which wraps up a set of changes you have made and asks you to leave a note that will help give context to any user who looks at it.

GitHub is a great place to store code because it's efficient and practical. It also makes it possible to access the code from anywhere and share it with everyone.
